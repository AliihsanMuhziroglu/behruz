# Мир приключений Бехруза 🚀

Интерактивный персональный сайт для Бехруза — игры, задания, викторины и веселье!

## Технологии

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Запуск

```bash
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

```bash
npm run build
npm start
```
